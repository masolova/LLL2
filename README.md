# Latvian SRS App

## GitHub Pages
1. Upload all files from this folder to the root of your repository.
2. Open the repository on GitHub.
3. Go to Settings -> Pages.
4. In 'Build and deployment', choose 'Deploy from a branch'.
5. Select branch: `main`, folder: `/ (root)`.
6. Click Save.
7. Wait 1-3 minutes and open the public URL GitHub shows on the same page.

The app entry point is `index.html`. The phrase dataset is in `phrases-500.json.js`.
